# star-formation
Files, scripts, and assorted data used in the radiative transfer star formation modeling project

/utilities/ will contain scripts of general usefulness. If I haven't modified the README at all over the next few months, and eventually someone inheriting the project reads this: start here!

/make_files/ contains files needed for generating runs of radmc3d, including the underlying physical dust density profiles and simulation grids 

/model_parameters/ will contain files describing the inputs to the simulations
